Use and configure
=================

Here's how to use and configure the TOC file.

# Installation

You can install `sphinx-globaltoc` with `pip`:

```bash
pip install sphinx-globaltoc
```

# Usage

In your `conf.py` configuration file, add `sphinx_globaltoc`
to your extensions list.

E.g.:

```{code-block} python
extensions = [
    ...
    'sphinx_globaltoc'
    ...
]
```
